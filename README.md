🧮 BMI Calculator

A simple and user-friendly Body Mass Index (BMI) Calculator built using HTML, CSS, and JavaScript.

This tool allows users to quickly check whether they fall under Underweight, Normal, Overweight, or Obese categories based on their height and weight.

⭐ Features

Easy input for height and weight
Instant BMI calculation
Clear result with BMI category
Clean and responsive UI

🚀 Technologies Used

HTML
CSS
JavaScript

🎯 Purpose
This project helps users understand their BMI value and maintain awareness of their health status
